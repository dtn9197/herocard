What is this?

These are called web components!!! Web components are custom re-usable HTML markup that can be used across different webpages. The source code for those components is written once and can be re-used everywhere. Their CSS properties are encapsulated in the Shadow DOM which is isolated from the main DOM. This mean that they won't be affected by regular css selectors.

Use Case

Card design pattern is very common in modern websites. They are used mostly to display information about people. This picture here is the executive team from Walmart's website. In this instance, leveraging web components will definitely boost productivity by saving development time.
